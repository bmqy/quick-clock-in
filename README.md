# 快速打卡
### quick-clock-in

## 介绍
JSBox脚本应用：快速发送预置好的打卡信息到输入框

## 配置项
- 部门
- 姓名

- 上午上班时段
- 下午上班时段
- 序号格式
- 显示年份
- 显示秒数
- 日期补零
- 时间补零(默认开启)

## 注意事项
1. 打卡序号需手动输入；
2. 请务必开启JSBox键盘的“允许完全访问”权限，否则使用过程中会有问题；
